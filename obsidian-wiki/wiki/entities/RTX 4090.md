# RTX 4090

RTX 4090-class hardware is the operator's 24GB development tier. It is strong
enough for serious WanGP targets, including LTX-2.3 distilled and explicit
short LTX-2.3 dev quality tests, but it should not inherit the 96GB+ high-memory
proof defaults.

## Operator Default

- Model target: `ltx23-distilled-22b` for iteration.
- Quality target: explicit `ltx23-dev-22b` after dry-run.
- Runtime flags: `sdpa`, `profile 3`, compile off, TeaCache `2.0` for balanced
  iteration and optional for quality tests.
- Escalation: try Sage/Sage2 only after local dependency compatibility is proven.

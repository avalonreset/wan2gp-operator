# Support

## Getting Help

- **Bug reports**: [Open an issue](https://github.com/avalonreset/wan2gp-operator-skill/issues) using the bug report template
- **Feature requests**: [Open an issue](https://github.com/avalonreset/wan2gp-operator-skill/issues) using the feature request template
- **Questions**: [Open an issue](https://github.com/avalonreset/wan2gp-operator-skill/issues) with the question label

## Before Opening an Issue

1. Check existing [open issues](https://github.com/avalonreset/wan2gp-operator-skill/issues) for duplicates
2. Run `python scripts/wan2gp_operator.py diagnose` to collect diagnostic info
3. Include the Wan2GP version, GPU model, and VRAM in your report

## Security Issues

Do not open a public issue for security vulnerabilities. See [SECURITY.md](SECURITY.md) for the reporting process.
